# Branding

Branding allows administrators to customize the visual appearance of the Obot platform. Access this page from **Branding** in the sidebar.

## Theme Colors

Customize the color scheme for both light and dark modes. Each scheme has configurable colors for:

- **Primary**: The main accent color used for buttons, links, and highlights
- **Background**: The page background color
- **Primary Text**: The main text color
- **Secondary Text**: The color for less prominent text
- **Surface 1, 2, 3**: Colors for cards, panels, and layered UI elements

Toggle **Live Preview Mode** to see changes in real-time before saving.

## Icons & Logos

Upload custom icons and logos to match your organization's branding:

- **Default Icon**: The standard icon shown in the UI
- **Error Icon**: Icon displayed for error states
- **Warning Icon**: Icon displayed for warning states
- **Full Logo**: The main logo shown in the navbar (separate versions for light and dark schemes)
- **Full Enterprise Logo**: Logo used when running enterprise version of Obot
- **Full Chat Logo**: Log shown in the navbar when using Obot Chat

## Restoring Defaults

Click **Restore Default** to reset all preferences to the original Obot theme.
